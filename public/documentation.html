Welcome to Football-API documentations V1. This API based on Fotmob's API which provide large amount of football data in real-time.

# API Endpoint

This is base URL that you will use for using this API. You have to change the API Endpoint to the API Endpoint you want to use.

Base URL:

https://api.aditiapras.dev/v1/{API_ENDPOINT}?apikey={YOUR_API_KEY}

GET

# Get All Leagues

Endpoint:

/v1/allLeagues?apikey={YOUR_API_KEY}

Example Request:

https://api.aditiapras.dev/v1/allLeagues?apikey={YOUR_API_KEY}

Example Response:

{
    
    "responses": {
      "favourite": null,
      "popular": [
        {
          "id": 47,
          "name": "Premier League",
          "localizedName": "Premier League"
        },
        {
          "id": 42,
          "name": "Champions League",
          "localizedName": "Champions League"
        }
        ]
    }
}

# Get All Matches

Endpoint:

v1/matches?apikey={YOUR_API_KEY}&date={date}

Parameter:

date={date} Format: "YYYYMMDD"

Example Request:

https://api.aditiapras.dev/v1/matches?apikey={YOUR_API_KEY}&date=20230724

Example Response:
